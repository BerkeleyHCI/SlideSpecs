# SlideSpecs

### startup

cloning:

    git clone https://github.com/berkeleyhci/peer-feedback && cd app
    meteor npm install

running:

    open http://localhost:3000
    meteor

deploying

    ssh bayscope # bayscope2.eecs
    cd Desktop/peer-feedback
    git pull
